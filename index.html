Blank 0.1 (Dirty Bomb)

Blank es un sistema de diseño y desarrollo front-end modular. Está diseñado para agilizar los procesos de wireframing, diseño y maquetación en el area de experiencia de America Móvil.

Grids (_grid.sass)

La retícula es la base estructural del framework. Las clases están construidas y declaradas en la hoja de estilos parciales _grid.sass Las clases tienen un valor inicial de 100% width y responden a un solo breakpoint (escalable) donde se dividen entre 12 columnas (12/100%). Su uso dentro del contenedor row deberá sumar 12 unidades ej. 12, 6/6, 4/4/4, 3/3/3/3 etc.
Markup
<div class=".row">
  <div class=".grid-1">Content</div>
  <div class=".grid-11">Content</div>
</div>

Tipografía (_type.sass)

La hoja de estilos parcial _type.sass contiene estilos base para los elementos de texto más comunes; Titulares, parrafos y cuerpo basados en los radios de una escala músical y un ritmo vertical relacionado a la misma escala. El cuerpo base de la tipografía ha sido establecido con un valor inicial de 19px. Usamos Gridlover para generar la escala y el ritmo.

Forms (_forms.sass)

La hoja de estilos parcial _forms.sass contiene estilos base para formularios con diferentes modificadores dependiento de las necesidades de comportamiento de los formularios. El valor inicial de los formularios es display: block y 100% width (mobile first) además de .form--inline para los formularios en linea y .form--stack para formularios agrupados en bloques.
Markup
<form class=".form .form--inline .form--stack">
  Default form
</form>
Default Form
  • .form Clase para formularios convencionales (100% width/ Display: block)
Inline Form
  • .form--inline Clase para formularios en línea

Buttons (_buttons.sass)

Los botones están construidas con clases separadas a los formularios y se puede usar en inputs, anchors y buttons. Todos los botones tienen un valor inicial de 100% width y display block a excepción del modificador de botones de tipo --inline. Se pueden añadir modificadores extras de apariencia utilizando la sintaxis BEM para modificar bloques: --modificador.
Markup
<button class=".button">Button</button>
<input type="submit" class=".button">
<a href="#" class=".button">Link</a>
Default
  • .button Clase para construir botones
  • .button--secondary Clase para construir botones con estilos secundarios
Inline
  • .button--inline Clase para construir botones en línea
